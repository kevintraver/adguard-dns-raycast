# Changelog

## [Initial Release] - {PR_MERGE_DATE}

### Added

- **AI-Powered Troubleshooting**: Natural language interaction via `@adguard-dns` mention in Raycast AI
- **Query Log Analysis**: Fetch and analyze blocked DNS requests with configurable time windows
- **Domain Unblocking**: Unblock domains with native confirmation dialog for safety
- **Root Domain Prioritization**: Automatically groups subdomains and unblocks at root level for broader coverage
- **Blocked DNS Requests Command**: Visual list UI for reviewing and unblocking domains
- **Device Name Display**: Shows which device triggered each blocked request
- **Browser Integration**: Open query logs in AdGuard DNS dashboard with Cmd+O
- **Automatic Token Refresh**: Handles AdGuard DNS API authentication seamlessly
